## Sumar

- Aplicație gratuită, open-source
- Afișează pe o hartă openstreetmap toate adăposturile de protecție civilă
- Oferă filtre suplimentare comparativ cu alte aplicații existente

## Date

- ISU: https://igsu.ro/resources/528ae255-7b59-4c84-bbed-252a49e871af.pdf
- fișier JSON disponibil în /data/allShelters.json

## Colaboratori

- Orice colaborator serios poate crea un pull request cu features (filtre etc.)
- Unele date din JSON, deși parsate corect direct de la sursă, sînt greșite (de exemplu, unele coordonate geografice). Orice ajutor pentru a corecta prostiile celor de la ISU sînt binevenite
